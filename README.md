# cbt

This repository contains code referenced in the paper:

**Dynamic Mission Reconfiguration and Knowledge Injection for Autonomous Vehicles Decision-Making Frameworks during Field Operations**

Here you can find the work presented for the paper.

In `bt_framework`, you can see all the material related to the framework itself. In `BB_BR_MPC` you can find the data related to the scenario used to showcase the work.

The repository is functional and can use some polishing. Every contributor is welcome.
